# Apex Motors Landing

Современный адаптивный landing page для премиального автосервиса.

## Стек

- Next.js App Router
- React + TypeScript
- Tailwind CSS
- Framer Motion
- Lucide Icons
- SEO metadata, Open Graph, JSON-LD, robots и sitemap

## Запуск

```bash
npm install
npm run dev
```

Для production-сборки:

```bash
npm run build
npm run start
```
