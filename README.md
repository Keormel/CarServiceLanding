# Aleks-Blitz Landing

Landing page modern și adaptiv pentru service-ul auto premium Aleks-Blitz.

## Stack

- Next.js App Router
- React + TypeScript
- Tailwind CSS
- Framer Motion
- Lucide Icons
- SEO metadata, Open Graph, JSON-LD, robots și sitemap

## Rulare

```bash
npm install
npm run dev
```

Pentru build production:

```bash
npm run build
npm run start
```
